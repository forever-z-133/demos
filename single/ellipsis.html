xBEM是一种继有限元法之后发展起来的一种新数值方法，与有限元法在连续体域内划分单元的基本思想不同，边界元法是在定义域的边界上划分单元，用满足控制议程的函数去逼近边界条件。所以边界元法与有限元相比具有单元的未知数少，数据准备简单等优点。但用边界元法解非线性问题时，遇到同非线性项相对应的区域积分，这种积分在奇异点附近有强烈的奇异性，使求解变得困难。

方法：float + 隐藏与遮挡
通过修改 height(1.5em的倍数) 来控制行数范围

缺点：背景色非白色时很尴尬，需要多一层嵌套

.text3 {
    overflow: hidden;
    height: 3em;
    line-height: 1.5em;
}
.text3:before {
    content: "";
    float: left;
    width: 1px;
    height: 100%;
}
.text3 > *:first-child {
    float: right;
    width: 100%;
    margin-left: -5px;
}        
.text3:after {
    content: "\02026";
    position: relative;
    float: right;
    top: -25px;
    left: 100%;
    width: 3em;
    margin-left: -3em;
    padding-right: 5px;
    text-align: right;
    background: linear-gradient(to right, rgba(255, 255, 255, 0), white 50%, white);
}
