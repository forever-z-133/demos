整理一些常见的布局

上下左右居中 - 定位法

.demo1 .center-xy-box {
    position: relative;
}
.demo1 .center-xy-item1 {
    position: absolute;
    top: 50%; left: 50%;
    transform: translate(-50%,-50%);
    /* 定宽定高按下策 */
    /*width: 100px; height: 30px;
    margin: -15px 0 0 -50px;*/
}

要求父级 relative，可使用 translate 居中，也可使用 margin 负值居中（有定宽定高的情况下），可以不定宽高。

我是子级

如果是定宽定高还有另一种定位写法：

.demo1 .center-xy-item2 {
    position: absolute;
    top: 0; bottom: 0;
    left: 0; right: 0;
    width: 100px; height: 30px;
    margin: auto;
}
我是子级

上下左右居中 - table 法

.demo2 .center-xy-box {
    display: table;
}
.demo2 .center-xy-item {
    display: table-cell;
    vertical-align: middle;
}

要求父级为 display:table。以上下居中见长，局限和搭配都挺多。如果仅为文字且不考虑背景色，父子两级即可，非常好用。子级不建议多个。

我是子级

如果仅有文字

若需套入更复杂结构，推荐三级嵌套，第三级即可上下居中。

.demo2 .others {
    background: #fff;
    /* 若需定宽，需另行左右居中*/
    width: 100px; margin: 0 auto;
    /* 若 inline-block 需父级 text-align:center */
}
<div class="center-xy-box center-xy-box2">
    <div class="center-xy-item">
        <div class="others">本内部结构就可以很复杂了</div>
    </div>
</div>
本内部结构就可以很复杂了

上下左右居中 - float 法

.demo3 .center-xy-box:before {
    content: "";
    height: 50%;
    margin-bottom: -15px;
}
.demo3 .center-xy-item {
    height: 30px;
    /* 如果要定宽则需要另行左右居中 */
    width: 100px;
    margin: auto;
}

要求定高，且需清除浮动。用父级的 :before 撑起一半的高度，然后 margin-bottom 负值拉回到上下居中，所以还需要该父级内没有其他元素

我是子级

上下左右居中 - flex 法

.demo4 .center-xy-box {
	display: flex;
	align-items: center;
	justify-content: center;
}

要求父级为 flex 容器。子级有时宽度会有 BUG，所以推荐最好定宽。

我是子级
